Technical Description


Browser Testing

My site is tested and working on Google Chrome and Firefox. I did not test in IE because lol IE. My site is also fairly mobile friendly due to responsive design where relative sizes(%) were used instead of fixed sizes (px).

Alignment

The entire page is in a div called container. Placing the whole site in a container div makes it easier to align and manage the padding of the page

Navigation

Navigation on the site is done through the menu bar. The bar is able to change colour when the mouse is over it due to the hover selector a:hover

CSS Summary

A single CSS stylesheet was used to style this website. CSS dictated how everything was positioned and how it appeared.

CSS Selector Category How was this used?
#container ID Keeps the entire page in a container div, making setting margins more uniform. 
#menu ID Allows the menu bar to be styled without affecting other elements
#head ID Used to style the header bar separate from the menu bar
#content ID Aided to style the body text/headings of the website
body Element Used to set background color and overreaching fonts
h1,h2,h3 Element These were styled within IDs, allowing them to be diffrent based on the div they were in
@import Rule Used to import a font from Google Fonts

Aesthetics

Colour Scheme

The background of this website is a light pink colour, complementing the dark gray div elements. I like this colour as it is a fresh change from the plain white backgrounds, while not being as cluttered as an image background. For the text, white was used, providing contrast and nice readability even during the late hours of the night.

Fonts

The font I chose to use was Open Sans, a sans-serif font from Google Fonts. I like this font because it gives a more modern feel to the website. The font was made more aesthetic by changing the font weight and letter spacing for headings.

Accessiblity

The website is also suitable for people who suffer from colourblindness. From using contrasting colours, text and other elements are easier to see. Additionally, text and other elements on this website are quite large, making them easier to read for the vision impaired.

The website is suitable for users lacking fine motor control or any other disability that prevents the use of a mouse. The entire website can be accessed using the keyboard, using the shift key to select menu items and the space key to scroll

The site is suitable for people with hearing disabilities as there is no audio on the site.
